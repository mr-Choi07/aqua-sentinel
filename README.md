# 어장지킴이 (Aqua Sentinel)
### 고수온·적조 조기경보 + 피해 증빙 자동화 AI

> 2026 부산·울산·경남 AI 활용 아이디어 공모전(나는 Solo AI) 출품작

## 1. 프로젝트 소개

통영·거제·남해·고성·하동의 해상 가두리·패류 양식 어가는 매년 여름 고수온·적조·빈산소로
막대한 피해를 입고 있습니다. 그러나 국립수산과학원의 특보는 '해역 단위'로만 발령되어
개별 어장·어종에 맞는 구체적인 행동 지시로 이어지지 못하고, 피해 신고와 증빙도 여전히
수작업에 의존하고 있어 정당한 보상이 새고 있는 실정입니다. **어장지킴이**는 공공 API로
실시간 수온·기상 데이터를 수집해 어장·어종별 위험도를 예측하고, LLM 기반 대응 코치와
폐사 사진 기반 신고서 자동생성 기능까지 제공하는 웹 대시보드로 이 문제를 해결합니다.

## 2. 왜 이 지역에 필요한가

경남지역 고수온 피해는 2021년 213개 어가·116억 5,900만 원에서 2023년 322개 어가·
207억 2,200만 원, 2024년 925개 어가·659억 2,200만 원(어류 2,396만 9천 마리 포함)으로
3년 새 어가 규모 4배, 피해액 약 5배 증가했습니다.

> 출처: 경남신문(2025.3.4) 종합 보도, 해양수산부 보도자료(mof.go.kr) 재난지원금 지급
> 현황 기준. 어가 수·피해액은 언론 보도 기준이며, 최종 확정 수치는 해양수산부/경상남도
> 공식 통계를 별도 확인 권장.

이러한 피해 규모에도 불구하고 해역 단위 특보는 개별 어장의 즉각적인 대응 판단에는
한계가 있고, 피해 신고·증빙 절차의 수작업 의존은 보상 누락으로 이어지고 있어 이 지역에
특화된 조기경보 및 증빙 자동화 솔루션이 필요합니다.

## 3. 주요 기능

1. **어장 맞춤 위험 예측** — 공공 API의 실시간 수온·기상 데이터를 기반으로 어장 위치와
   양식 어종 특성을 반영한 개별 위험도를 산출합니다.
2. **LLM 기반 대응 코치** — Anthropic API 기반 RAG로 현재 위험 상황에 맞는 구체적인
   대응 행동(가두리 이동, 산소 공급 등)을 안내합니다.
3. **피해 증빙 자동화** — 폐사 사진을 멀티모달 AI로 분석하고, 신고에 필요한 증빙 서류를
   PDF로 자동 생성합니다.

## 4. 기술 스택

- **Python 3.11+**
- **Streamlit** — 대시보드/UI
- **SQLite** — 초기 DB (추후 필요 시 Postgres 전환 가능하도록 설계)
- **pandas** — 시계열 데이터 처리
- **requests** — 공공데이터포털 API 연동
- **Anthropic API** — RAG 기반 대응 코치, 멀티모달 폐사 사진 분석
- **fpdf2** — 신고서 PDF 생성
